Detailed Description

The Controller class for the Tic-Tac-Toe application.

CGame.h

Created on: 01-Dec-2025 Author: giris

This class acts as the central engine (Controller in MVC). It manages the lifecycle of the game, including the Board (Model), the Display (View), and the Players. It controls the game loop, turn switching, and result checking.
