Detailed Description

Represents the Tic-Tac-Toe game board data and logic.

CBoard.h

Created on: 29-Nov-2025 Author: giris

This class manages the internal 1D vector representation of the 2D grid, handles coordinate validation, and implements the algorithms to detect win conditions (Rows, Columns, and Diagonals).
