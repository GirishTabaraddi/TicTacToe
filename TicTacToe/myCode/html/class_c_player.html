Detailed Description

Abstract Base Class representing a participant in the game.

  • This class defines the interface for any entity that can play Tic-Tac-Toe, whether it is a Human or a Computer. It implements the Strategy Pattern via the pure virtual function decideMove().

virtual Coordinates CPlayer::decideMove ( const CBoard &  board)

Calculates or retrieves the coordinates for the next move.

  • This is a Pure Virtual Function that must be implemented by derived classes.
  • For CHuman: It prompts the user for input via cin.
  • For CComputer: It calculates the best move via simulating the moves or Random logic.
