Detailed Description

Represents an Smart Computer player in the Tic-Tac-Toe game.

CComputer.h

Created on: 1 Dec 2025 Author: GTABARA

  • This class uses the algorithm of calculating optimal moves for 3x3 boards to play smartly and falls back to random selection for larger boards to save computation time.
